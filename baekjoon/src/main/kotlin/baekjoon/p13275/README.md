문제
문자열 S의 부분 문자열 중에서 팰린드롬인 것 중 가장 긴 것의 길이를 구하는 프로그램을 작성하시오.

입력
첫째 줄에 문자열 S가 주어진다. S는 알파벳 소문자로만 이루어져 있으며 길이는 1보다 크거나 같고, 100,000보다 작거나 같다.

출력
가장 긴 팰린드롬 부분 문자열의 길이를 출력한다.

예제 입력 1
abcd
예제 출력 1
1
예제 입력 2
abab
예제 출력 2
3
예제 입력 3
dcabccd
예제 출력 3
2
예제 입력 4
abcba
예제 출력 4
5
예제 입력 5
aaaaa
예제 출력 5
5
